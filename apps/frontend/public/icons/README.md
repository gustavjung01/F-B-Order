# PWA icons

Đặt logo PWA thật tại đây khi có brand/icon chính thức.

File cần có:

- `apple-touch-icon.png` - 180x180
- `icon-192.png` - 192x192
- `icon-512.png` - 512x512
- `maskable-512.png` - 512x512, chừa safe area

Không dùng ảnh người/avatar làm icon app. Nên dùng logo rõ, nền đặc, nhìn tốt trên màn hình điện thoại.
